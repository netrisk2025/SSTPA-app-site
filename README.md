# SSTPA Tools Website

A Vercel-ready promotional website for **SSTPA Tools**.

## Stack

- Vite
- React
- TypeScript
- Locally bundled open-source fonts via Fontsource:
  - Source Sans 3
  - Cormorant SC
  - JetBrains Mono

## Local development

```bash
npm install
npm run dev
```

## Production build

```bash
npm run build
```

The production output is written to `dist/`. Vite copies the white paper from `public/files/SSTPA-Tools-White-Paper-v2.docx` into the deployed static output.

## Vercel deployment

This project includes `vercel.json` with:

- build command: `npm run build`
- output directory: `dist`
- framework: `vite`

After a Vercel personal access token is available, deploy from this directory with:

```bash
npx vercel --prod --yes --token "$VERCEL_TOKEN"
```

If this is the first deploy for the project, Vercel may need project linking metadata. The `.vercel/` directory is intentionally ignored and should be generated by the Vercel CLI.
